# servo command - Arduino Skech

Test for Continuous Rotation Servo <br/>
Command specify forward, backward or etc <br>

### Usage
format : s + three digits <br>
example <br>
- s000 : clockwide full speed <br>
- s090 : stop <br>
- s180 : anticlockwide full speed <br>
